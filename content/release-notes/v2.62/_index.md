### New "User Login" Report has Been Created

**CACTWO-5478** **{{< rawhtml >}}<span style="color:#1F497D">(Enhancement)</span>{{< /rawhtml >}}**

A new report titled "User Login" has been created.  This will show the login/logout activity of the user. If an inactivity reason was recorded by the user, then the prior logout will be listed as ‘inactivity’ instead of logout and the inactivity reason will be recorded in its own column.  Date range available is 31 days, the report will never show the current day’s activity.
